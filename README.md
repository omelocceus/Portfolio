Rock Paper Scissors Portfolio Project
This is a simple portfolio website that includes a Rock Paper Scissors game. I built this project as a beginner while learning the basics of HTML, CSS, and JavaScript. The goal of this project is to practice web development fundamentals and create something interactive.
Project Description
This project is a basic portfolio page with a Rock Paper Scissors game. The user plays against the computer, which randomly chooses rock, paper, or scissors. The result of each round is shown on the screen. This project helped me understand how websites work and how JavaScript adds interactivity.

Technologies Used:

HTML – for the structure of the website
CSS – for styling and layout
JavaScript – for game logic and user interaction

Features Implemented: 

Required Features
Rock, Paper, and Scissors buttons
Random computer choice
Win, lose, or draw result
Display of user and computer choices
Additional Features
Simple portfolio layout
Easy-to-use buttons
Clean and readable code
